9 · Практика и проверка

Сделайте на своей задаче

Возьмите один свой обезличенный рабочий документ (регламент, короткую инструкцию, выдержку из положения) и задайте по нему вопрос-ответ по промпту B1. Проверьте хотя бы один ответ по первоисточнику вручную.

⚙ Для переноса в платформу Актион
Здесь платформа подключает отправку куратору — проверка №22 (см. паспорта шагов). В демо-комплекте вместо реальной отправки — локальное скачивание .txt с текстом задания, ничего никуда не отправляется.
⬆️ Ассистент закрепил навык «вопрос-ответ по своим документам». Что забрать: загруженный документ превращает ответ «из головы» в ответ по источнику; требование ссылки на пункт в промпте обязательно; «в документе нет» — правильный ответ, а не провал; много документов — уже задача «настоящего» RAG для ИТ.
